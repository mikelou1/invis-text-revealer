# Invisible Text Revealer

A lightweight Chrome extension that reveals hidden or invisible text on any webpage.

**NOTE this is unlikely to work on documents such as pdf files on chrome**

## Features

- Highlight mode (Highlights text with red)
- Show mode (Shows text but does not highlight)
- OFF (Acts as if extention is not enabled)

## How to Use

1. Load unpacked into chrome://extentions
2. Select settings by clicking the jigsaw -> Invisible text revealer
3. Automatically highlights

## Use Cases

- Identifying hidden text (i.e. font 0)
